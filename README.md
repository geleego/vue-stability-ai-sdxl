# vue-stability-ai-sdxl
StabilityAI REST API를 활용한 SDXL Txt2Img Vue 프로젝트
- 개발 환경: Vue

![result](https://github.com/geleego/vue-stability-ai-sdxl/blob/main/doc/thumbnail.png "result")

## 프로젝트 세팅 Node 버전
```
v16.14.2
```

## 프로젝트 설치 명령어
```
npm install
```

### 개발 서버 가동 명령어
```
npm run serve
```

### 프로젝트 배포용 파일 생성 명령어
```
npm run build
```
